Google Sheets 連結
如何連結 Google Sheets

1. 開啟一個 Google 試算表
2. 上方選單：擴充功能 → Apps Script
3. 貼上以下程式碼並儲存：

function doPost(e) {
  var data = JSON.parse(e.postData.contents);
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(['序號','出貨日期','出貨時間','備註']);
  }
  sheet.appendRow([data.sn, data.date, data.time, data.note||'']);
  return ContentService.createTextOutput('ok');
}
4. 點「部署 → 新增部署」
5. 類型選「網頁應用程式」
6. 執行身分：我，存取權：任何人
7. 複製網址貼到上方欄位
